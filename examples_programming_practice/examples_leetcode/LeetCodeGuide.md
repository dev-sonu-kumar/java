## üí° Problem-Solving Patterns

### **Two Pointers Pattern**
- **Use When**: Array/string problems with pairs or subarrays
- **Examples**: Two Sum, 3Sum, Container With Most Water
- **Time Complexity**: Usually O(n) or O(n¬≤)

### **Sliding Window Pattern**
- **Use When**: Contiguous subarray/substring problems
- **Examples**: Longest Substring, Maximum Subarray
- **Time Complexity**: Usually O(n)

### **Fast & Slow Pointers**
- **Use When**: Linked list cycle detection, finding middle
- **Examples**: Linked List Cycle, Middle of Linked List
- **Time Complexity**: O(n)

### **Merge Intervals Pattern**
- **Use When**: Overlapping intervals problems
- **Examples**: Merge Intervals, Insert Interval
- **Time Complexity**: O(n log n)

### **Tree Traversal Pattern**
- **Use When**: Tree/graph problems
- **Examples**: All tree traversals, Path Sum
- **Time Complexity**: O(n)

### **Dynamic Programming Pattern**
- **Use When**: Optimization problems with overlapping subproblems
- **Examples**: Fibonacci, Coin Change, House Robber
- **Time Complexity**: Varies, often O(n) to O(n¬≤)
